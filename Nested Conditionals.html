JavaScript - Nested Conditionals

Things to Know

  • In nested conditionals, we check the statement on the basis of multiple conditions

Syntax

conditional condition1 { // code to be executed if condition1 is true conditional condition2 { // code to be executed if both condition1 and condition2 are true } }
